# Screenshots Guide

Please add your screenshots with the following names in this exact order:

## Required Screenshots:

### 1. `01-main-dashboard.png`
- **What to capture**: The main application view showing:
  - Post creation form at the top
  - Recent posts feed in the center
  - Karma leaderboard on the right side
  - User profile in the header
- **Recommended size**: 1920x1080 or similar wide format

### 2. `02-post-with-image.png`
- **What to capture**: A single post that includes:
  - User avatar and name
  - Post content with an uploaded image
  - Like button with count
  - Comment button with count
  - Timestamp
- **Recommended size**: 800x600 or similar

### 3. `03-comments-modal.png`
- **What to capture**: The comments modal/popup showing:
  - Original post at the top
  - Comment creation form
  - Threaded comments with replies
  - Like buttons on comments
  - User avatars and names
- **Recommended size**: 1200x800 or similar

## Tips for Great Screenshots:

1. **Use a clean browser**: Clear any browser extensions or bookmarks from view
2. **Good lighting**: Ensure the screenshots are clear and well-lit
3. **Consistent sizing**: Try to maintain similar dimensions for all screenshots
4. **Show real data**: Use actual posts and comments, not placeholder text
5. **Highlight features**: Make sure key features like likes, comments, and karma are visible

## File Format:
- Use PNG format for best quality
- Compress images if they're too large (keep under 2MB each)
- Ensure images are clear and readable

Once you add the screenshots, they will automatically appear in the main README.md file!
